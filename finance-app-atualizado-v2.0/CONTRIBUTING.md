# Contributing to FinançasPro

Obrigado por estar interessado em contribuir! Por favor, siga as diretrizes abaixo:

## 🐛 Reportar Bugs

- Use a aba "Issues" para reportar bugs
- Descreva o comportamento esperado vs. o comportamento real
- Inclua screenshots ou logs se possível

## 🎯 Sugerir Melhorias

- Use a aba "Discussions" para sugestões
- Explique o caso de uso e os benefícios
- Forneça exemplos de como a funcionalidade funcionaria

## 📝 Pull Requests

1. Fork o repositório
2. Crie uma branch para sua feature (`git checkout -b feature/AmazingFeature`)
3. Commit suas mudanças (`git commit -m 'Add some AmazingFeature'`)
4. Push para a branch (`git push origin feature/AmazingFeature`)
5. Abra um Pull Request

## 📋 Padrões de Código

- Use componentes funcionais React
- Siga as convenções de nomenclatura existentes
- Adicione comentários para lógica complexa
- Use Tailwind CSS para styling

---

Valorizamos todas as contribuições! 🙌
